What are Implicit Facts?

Implicit facts are relationships and information that are not directly stored in the graph but can be inferred from existing data. For example:

  • If A is parent of B, and B is parent of C ‚Üí A is grandparent of C
  • If X is connected to Y, and Y to Z ‚Üí X is indirectly connected to Z
  • Characters with names in multiple languages
  • Type inheritance chains

Technical Details

These queries use SPARQL 1.1 with property paths and inference capabilities. The data is fetched from the Fuseki triplestore using the http://localhost:3030/tolkienKG/sparql endpoint. Implicit facts are computed at query time, demonstrating the power of semantic reasoning.
